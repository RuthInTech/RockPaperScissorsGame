# 🪨📄✂️ Rock Paper Scissors Game

A simple Rock, Paper, Scissors game built with Python where the player competes against the computer. The computer randomly selects a choice, and the program determines the winner based on the classic game rules.

## 🎮 How It Works

* The player chooses one option:

  * 🪨 Rock
  * 📄 Paper
  * ✂️ Scissors

* The computer randomly selects its choice using Python's `random` module.

* The game compares both choices and displays the winner.

* Players can continue playing multiple rounds.

* The score is tracked throughout the game.

## ✨ Features

* ✅ Player vs Computer gameplay
* ✅ Random computer choices
* ✅ Multiple rounds
* ✅ Score tracking
* ✅ Input validation
* ✅ Simple command-line interface
